# GitHub Cards
A web page that populates with GitHub profile cards after entering a GitHub username. 

*This project was a coursework assignment from Pluralsight's "React: Getting Started" course*

# Technologies Used
* React.js
* JavaScript
* HTML/CSS
* Semantic UI
